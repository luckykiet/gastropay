## Instalation

1. Install dependencies

```
npm i -g yarn npm-run-all
yarn install
```

2. Set up .env file in _client_ folder from **.env.example** file
3. Run app

```
yarn start-local
```

4. Deployment on netlify

```
yarn deploy-prod
```
